Documentation site for [thinodium](https://github.com/hiddentao/thinodium), viewable 
live at [hiddentao.github.io/thindoium](https://hiddentao.github.io/thindoium).

## Building

To build assets:

```bash
$ npm install
$ npm run build
```

To watch for changes and rebuild accordingly:

```bash
$ npm run dev
```
